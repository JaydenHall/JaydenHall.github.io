The Dom

The DOM ( Document object model ) is an object orientated representation of your HTML document. A webpage is a document not matter if it is opened in a browser or in a code editor. We can use JavaScript to modify your HTML code via the DOM.

Functions, Objects & Arrays

There are two ways to access the properties of an object: the dot notation (.) and the bracket notation ( [ ] ) The Dot notation is what you use when you know the name of the property you’re trying to access ahead of time.

Let myObject = {

property1: “value1”

property2: “value2”

}

Let propertyValue = myObject.property1

We can access the data from arrays using square brackets. We Choose what entry in the array we would like by placing a number between the square brackets, the first element in an array has an index of 0.

Let array = [50,60,70];

array[0] would be equal to 50

Let data = array[1] would be equal to 60

Functions are a sequence of commands that take and input and give us an output. They are important because they are reusable and save us from repeating code.

Function addNumber( num1, num2 ) {

return num1 + num2

}
